# 🦺 VisionInspect AI

VisionInspect AI is an AI-powered workplace safety inspection platform designed to streamline the inspection process. By leveraging Google Gemini AI, the application analyzes workplace images to identify visible safety hazards, assess PPE compliance, and generate structured inspection reports with actionable recommendations. It helps safety professionals perform inspections more efficiently while promoting safer work environments.

## ✨ Key Features

* AI-powered workplace image analysis
* Automatic hazard detection
* PPE compliance assessment
* Risk evaluation and safety recommendations
* Structured inspection report generation
* Fast, responsive, and user-friendly interface

## 🛠️ Technology Stack

* **Frontend:** React, TypeScript, Vite, Tailwind CSS
* **AI Model:** Google Gemini API

## 🚀 Project Setup

### Clone the repository

```bash
git clone https://github.com/Hemalathaa2006/Vision_Inspect_AI.git
cd Vision_Inspect_AI
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file in the project root and add your Gemini API key:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the application

```bash
npm run dev
```

The application will start locally and can be accessed through the URL displayed in your terminal.

## 📖 Usage

1. Open the application in your browser.
2. Upload a workplace image for inspection.
3. Allow the AI to analyze the image.
4. Review detected hazards, PPE compliance, and safety recommendations.
5. Generate and view the inspection report.
